GitHub Repository Information

To get started, please provide the following information about the GitHub repository you want to export issues from. You will also need to provide a GitHub Access Token with the necessary permissions to read the repository's issues.




Using the Sign Out button removes the stored GitHub access token used by this app, if you have added one, from your local storage.

Label Filters
  • You must have entered a GitHub Access Token, Repository Owner, and Repository Name to use the label filters.
  • Use the "Load Repo Labels" button to populate the label options for the following fields.
  • Select multiple labels by holding Ctrl (Windows) or Cmd (Mac).
