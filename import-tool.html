GitHub Issue Importer

This tool is meant to help you import issues you exported with this app from one repo into another repo.

Import Configuration

Provide a dedicated GitHub token for import access, the target GitHub Issues API URL, and the exported issue files to import.

Use a token with read/write access to issues for the destination repository.

Removes the import token stored by this app from local storage.

Use the GitHub REST API issues endpoint (https://api.github.com/repos/owner/repo/issues) for the destination repository.

Select the extracted export folder that contains issue files and any assets directory.

Controls the delay between issue creation requests and retry aggressiveness. Slow recommended for imports of 80 or more.

Missing required information. Add an import token, a valid target issues URL, and an export folder before importing.
