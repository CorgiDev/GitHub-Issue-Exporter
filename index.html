GitHub Repository Information

To get started, please provide the following information about the GitHub repository you want to export issues from. You will also need to provide a GitHub Access Token with the necessary permissions to read the repository's issues.




Using the Remove Token button removes the GitHub access token used by this app from your local storage, if you added one.

Label Filters
  • All fields in the GitHub Repository Information must be filled in to populate label options.
  • Use the "Load Repo Labels" button to populate the label options. It will pull all the labels available in the selected repo.
  • You can select/unselect multiple labels by clicking on them.
  • Using the "Clear selected labels" buttons removes all the labels currently selected from its respective selection menu.
